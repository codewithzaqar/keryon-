# Kryon

**Version:** v0.0.1a
**Status:** Pre-Alpha  

Kryon is a modern, high-performance systems programming language focused on safety, concurrency, and zero-cost abstractions. It aims to provide memory safety without garbage collection through a strict ownership model, similar to Rust, but with syntax inspired by Python and Go for readability.

## Features (Planned)
- Static typing with type inference
- Ownership-based memory management
- Native concurrency primitives (async/await built-in)
- Zero-cost abstractions via compile-time generics

## Getting Started

### Prerequisites
- Python 3.10+

### Installation
```bash
pip install -e .
```
